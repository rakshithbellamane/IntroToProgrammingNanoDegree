How does the web work?

In simplest terms, web is made of clients, internet and servers.
Clients are the computers that users use to access internet and get information or do something.
Servers are computers that complete the requests that clients make. Usually, servers get requests, perform some computation and sends back responses.
Clients and Servers are connected by what we call Internet through which the requests and responses are routed.

What is HTML?

HTML or Hyper Text Markup Language is a document format to exchange information on internet.
Websites show information to users in browser as HTML document.
HTML is exchanged between client and servers using a protocol called HTTP or Hyper Text Transport Protocol.

What are the key components of HTML document?

HTML is primarily made up two sections:

  • <Head>: Identifies the Header of the HTML document. We usually provide metadata information here.
  • <Body>: Body contains the contents of the HTML document.
A simple HTML document structure is as follows:

 <!DOCTYPE>
  <html>
      <head>
      </head>
      <body>
      </body>
  </html>

What are elements and tags?

HTML document is made up of elements. These elements are identified through tags.
Some of the components of tags are (not all are mandatory. It depends on element type):

  • What element it is
  • Name of the element
  • Type of the element
  • Attribute of element
  • Value of the element
Example:<a href='https://www.nytimes.com'>The NewYork Times</a>

Elements can be inline elements and block elements.

Inline elements basically doesn't cause a new line. Ex: <img>.
Block elements cause a new line and form an invisible block around the element. Ex: <div>.

Elements can be styled using CSS or Cascading Style Sheets.

  • Inline:
